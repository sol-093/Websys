## Summary
- **Scope:** Teacher-friendly explanation of the current frontend design direction
- **Updated:** April 27, 2026
- **Theme:** Glass-style dashboard UX with green/teal visual identity

## Purpose
This document provides a presentation-ready explanation of the frontend concept, design choices, and practical value for school stakeholders.

## Audience
- Teachers and advisers evaluating the interface
- Administrators reviewing usability/readability
- Team members delivering product presentations

## Core Content

### One-Paragraph Pitch
This platform is a student organization management website that centralizes announcements, organization activity, and budget visibility in one dashboard experience. The interface is designed to be clean, readable, and presentation-friendly so staff can quickly understand key information without being overwhelmed.

### Design Rationale
- **Clarity first:** dashboard layout emphasizes fast scanning
- **Trust and transparency:** finance and activity sections are visible and structured
- **Low cognitive load:** consistent cards, spacing, and typography

### Visual Identity
- **Style:** glassmorphism-inspired layered surfaces
- **Color direction:** green/teal palette to suggest growth, balance, and trust
- **Typography:** clean sans-serif presentation for readability
- **Dual-mode support:** light and dark themes for viewing comfort

### Functional Experience Shown in Demo
- Home/login/register surfaces
- Role-aware dashboard structure
- Organization and admin management sections
- Search and modal workflows
- First-login onboarding walkthrough

### Short Spoken Script
> Our website is a student organization management system built for schools. It uses a modern glass-inspired layout with green and teal colors to make information clear, calm, and trustworthy. The goal is to help teachers and staff monitor organization activity and financial transparency in one readable dashboard.
